# Historical Data Template for 2026 Senate Elections

This file provides a template for entering historical data from the past 2 Senate elections for each state.

## Data Format Required

For each county in each state, please provide:
- **County Name**: Official county name (must match COUNTY-LISTS.md)
- **Dem Share**: Democratic vote share as decimal (e.g., 0.54 = 54%)
- **Turnout**: Total votes cast in that county for that election

## How to Calculate from Election Results

**Example:** If County X had these results in 2022 Senate race:
- Democrat: 45,000 votes
- Republican: 35,000 votes
- Total: 80,000 votes

Then:
- demShare = 45,000 / 80,000 = 0.5625
- turnout = 80,000

## Notes

- County names are in alphabetical order to match COUNTY-LISTS.md
- demShare should be calculated as: Democratic votes / Total votes
- turnout should be total votes cast (Dem + Rep + Other)
- If county had ranked-choice voting (Maine), use first-round totals
- All county names must exactly match those in COUNTY-LISTS.md
